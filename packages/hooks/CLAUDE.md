# Hooks Package - Claude Session Guide

## Never Edit `.claude/settings.json` Directly

Hook entries in `.claude/settings.json` (`PreToolUse`, `SessionStart`, `SubagentStart`, `WorktreeCreate`, etc.) are
generated by `dungeonmasterHooksCreatorTransformer` and written by `InstallCreateSettingsResponder`.

To add/change a hook, edit the transformer (and matcher/command definitions) — **do NOT hand-edit
`.claude/settings.json`**. See root `CLAUDE.md` ("Never Edit `.claude/settings.json` Directly") for the
build → `npm link --workspaces` → `npm run init` flow that regenerates settings for this repo.
